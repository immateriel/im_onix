ONIX Code Lists Issue 21, April 2013
List 179: Price code type
Value Description Notes
01 Proprietary A publisher or retailer’s proprietary code list which identifies particular codes with particular price points, price tiers or bands.
02 Finnish Pocket Book price code Price Code scheme for Finnish Pocket Books (Pokkareiden hintaryhmä). Price codes expressed as letters A–J in <PriceCode>.
