ONIX Code Lists Issue 21, April 2013
List 53: Returns conditions code type
Value Description Notes
00 Proprietary
01 French book trade returns conditions code Maintained by CLIL (Commission Interprofessionnel du Livre). Returns conditions values in <ReturnsCode> should be taken from the CLIL list.
02 BISAC Returnable Indicator code Maintained by BISAC: Returns conditions values in <ReturnsCode> should be taken from List 66.
03 UK book trade returns conditions code NOT CURRENTLY USED – BIC has decided that it will not maintain a code list for this purpose, since returns conditions are usually at least partly based on the trading relationship.
04 ONIX Returns conditions code Returns conditions values in <ReturnsCode> should be taken from List 204.
